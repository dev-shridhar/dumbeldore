# src/

Source code for the Dumbledore Telegram bot.

- `main.py` — Bot handlers and entry point
- `brain.py` — Versioned prompt loader
- `llm.py` — LLM client (Ollama + Groq fallback)
- `memory.py` — Per-group chat history
- `config.py` — Environment config
- `start_bot.py` — Entry point (loads .env)
- `prompts/` — Versioned prompt templates
